About Me

Who am I?

Hey there! I am Abby, a pixel wizard and a code conjurer based in Nairobi, Kenya.I am very passionate about learning new technologies and developing new projects. I really enjoy turning complex problems into simple, beautiful and responsive web interface designs by crafting digital experience from concept to code.

My Journey?

From tinkering with my first line of code to architecting robust web applications, my journey as a full-stack software engineer has been nothing short of exhilarating.My journey began with HTML and CSS, the modest canoe on the vast code ocean. At first, the new content and rules of coding were overwhelming but with practice, prayer, resilience and constant reassuarance from my family, I finally got the hang of things. Over time, I upgraded to sturdier vessels: JavaScript, Python and Bootsrap. I now steer projects with top notch confidence, weaving front-end sails and back-end anchors.

Mission?

  • Creating user-centric software solutions that enhance people's lives
  • To build seamless digital experiences
  • Solving real-world problems through code
  • Architecting robust and scalable systems
  • Promoting best practices, reducing technical debt and elevate the quality of software
